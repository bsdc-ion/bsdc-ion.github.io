Introduction

Welcome to the official documentation for Pawn Scripting, a lightweight and efficient scripting language designed for creating mods and custom server logic in multiplayer games. Whether you're a game modder, server administrator, or a developer looking to extend your game’s features, this guide will help you harness the power of Pawn to enhance your gameplay experience.


Pawn scripting is widely used in popular multiplayer games such as GTA: San Andreas Multiplayer (SA-MP) and Multi Theft Auto (MTA), enabling you to control game mechanics, interact with players, and modify virtually every aspect of the game world.


This documentation will guide you through the core concepts, syntax, and best practices of Pawn scripting, with step-by-step tutorials and examples to get you started. Whether you're building simple game modes, creating complex systems, or managing server events, you'll find everything you need to unlock the full potential of Pawn.

Player Spawn Event

This script handles what happens when a player spawns in the game.

public OnPlayerSpawn(playerid) {       SendClientMessage(playerid, 0x00FF00FF, "You have spawned!");       GivePlayerWeapon(playerid, 31, 100); // Gives player a pistol with 100 ammo       return 1; }
Creating a Custom Vehicle Spawn Command

This script lets players spawn a vehicle using a custom command.

CMD:car(playerid, params[]) {       new vehicleID;       vehicleID = CreateVehicle(411, GetPlayerPosX(playerid), GetPlayerPosY(playerid), GetPlayerPosZ(playerid), 0, 0, 0, -1);       SendClientMessage(playerid, 0x00FF00FF, "Your car has been spawned!");       return 1; }
